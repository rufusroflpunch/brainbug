BrainBug

The Brainf*ck Debugger

The usage is simple. Enter your Brainf*ck code to the left. To run the complete program, click Execute. To begin debugging the application, enter a breakpoint (*) anywhere in the program and click Debug. Upon encountering the breakpoint, the execution will pause and output a information about the application below. Use Continue to continue execution until the next breakpoint. Reset will begin again from the start. Lasty, Beautify will format your code in a more readable way.
